6B48: Compute and print a digit
Used by the routine at 6B21.
Input
A 48 (ASCII code for '0')
BC 10000, 1000, 100 or 10
DE Display file address
HL Number being printed
F Carry flag reset
Output
A 48 (ASCII code for '0')
F Carry flag reset
